# BlueSense4

![GitHub Logo](/img/side.png)

BlueSense4 - an extensible wearable motion sensing and IoT platform based on a ARM Cortex M4 (STM32L496).

Key features:
* Processor
  * STM32L496 ARM Cortex M4 processor
  * 1MB Flash, 320KB SRAM
  * 20MHz default – up to 80MHz
  * Built-in RTC
* Peripherals
  * ICM-20948 MPU
  * Built-in digital microphone
  * 500Hz motion sensing (raw + orientation)
  * Multimodal sensing: audio + motion + ADC
  * Logging and streaming
  * Extensible
  * Compatible with BlueSense2
  * Bluetooth 2.0
  * USB
  * SD card
  * EEPROM
  * DFU
